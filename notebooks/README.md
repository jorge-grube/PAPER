# Notebooks

This folder contains analysis notebooks for the empirical finance paper.

## Naming convention

- `01_*`: data audit and quality control
- `02_*`: exploratory diagnostics and transformations
- `03_*`: model preparation and feature construction
- `04_*`: estimation and robustness checks

## Current notebook

- `01_data_quality_audit.ipynb`: validates `reports/data_inventory.yml` and produces QA outputs in `reports/quality/`.
