## C. Definition of Label Agreement

The statistic is defined in `11_zew_swap_experiment.py` lines 686–691 as:

```python
lab_base = pd.read_parquet(lab_base_path)["regime_wf"].dropna()   # baseline labels
lab_zew  = pd.read_parquet(LAB_PATH)["regime_wf"].dropna()          # ZEW-swap labels
common_idx = lab_base.index.intersection(lab_zew.index)             # intersection of non-NaN dates
agree = (lab_base.loc[common_idx].values.astype(int) ==
         lab_zew.loc[common_idx].values.astype(int))
pct_agree = 100 * agree.mean()   # → 47.9339%
```

**Precise definition:**
- **What is compared:** Canonical integer state labels (0–3, VIX-ordered) from the baseline HMM (VSTOXX feature) vs. canonical integer state labels from the ZEW-swap HMM (ZEW_Germany replacing VSTOXX)
- **Date window:** Full walk-forward label history — all weeks where both label series are non-NaN (2007-10-19 to 2026-05-01)
- **Common dates:** 968 weekly observations
- **Frequency:** Weekly labels (every non-NaN walk-forward rebalance-eligible date), not rebalance-date-only
- **NaN handling:** Both series have NaN dropped before intersection; burn-in period excluded implicitly

**Note on Panel B window:** Over the Panel B window only (2010-10-15 to 2026-04-03, 812 dates), the agreement is 48.3% — 0.4 percentage points higher. The script computes and reports the statistic over the full walk-forward period (968 dates), not the Panel B window. The paper states "47.9%" without qualifying the window, which matches the script's full-window computation. This is the appropriate statistic since the HMM walk-forward begins in 2007, not 2010.

## G. Verification Command (Reproducible)

```python
import pandas as pd

baseline = pd.read_parquet('data/processed/regime_labels_wf_156.parquet')
zew      = pd.read_parquet('data/processed/model_improvement/regime_labels_wf_156_zew_swap.parquet')

b = baseline['regime_wf'].dropna()
z = zew['regime_wf'].dropna()

common = b.index.intersection(z.index)
agree  = (b.loc[common].astype(int) == z.loc[common].astype(int))

print(f"N = {len(common)}, Agreement = {100*agree.mean():.4f}%")
```
